**Description:**    

Responsive & SEO friendly tables with schema markup plugin for WordPress. 10+ beautiful designs. Unlimited tables. Easy to use. Shortcodes. it contains everything you will ever need to create awesome tables.

**Credits and fork information:**  
This plugin is a fork of Table Maker 1.6 created by Wpsoul. <br>
I chose to fork and rebuild this plugin because I use it so much for all of my websites. 
The only thing that matters for my websites is that they rank well in Google, therefor all plugins I use have to be completely and fully seo optimized & seo friendly.
Since I made the changes to the plugin anyway, I thought other people might find good use of it!
You can donate to and read more about the author Wpsoul [here](https://wpsoul.com/)
